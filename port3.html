Project

Project Description: This project is a banner design created to present the creator’s name and identity. It combines colorful typography with a patterned background to create a bold and eye-catching visual. A photo of the creator is included to establish a unique identity, along with a contact channel via Instagram (@phumiphat_2007). The highlight of this work is the use of contrasting color tones (green–orange–red), which enhances readability and draws attention to the main text.
